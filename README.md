# 🎓 Student Performance Prediction System

## 📌 Project Overview
This project predicts student performance using Machine Learning.

## 🚀 Features
- Predicts Pass / Fail
- Identifies at-risk students
- Helps teachers take corrective actions
- Data-driven academic insights

## 📊 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📁 Dataset
- student_academic_data.csv
- at_risk_students.csv

## 🧠 Models Used
- Logistic Regression
- Decision Tree
- Naive Bayes
- SVM

## 📌 Output
- Prediction of student result
- Visualization graphs
- At-risk students list

## 💡 Future Improvements
- Add web app (Streamlit)
- Improve model accuracy
- Real-time student monitoring
